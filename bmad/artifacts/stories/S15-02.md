**Epic:** Components

**Points:** 3

## User Story
As a developer, I want a `breadcrumb-pk` molecule that renders an accessible breadcrumb nav with ARIA landmark and separators, so that I can implement location indicators in UIs without custom markup.

## Acceptance Criteria

```gherkin
Given a breadcrumb-pk with slotted items
When the page loads
Then a nav[aria-label] with ol.breadcrumb__list is rendered

Given breadcrumb-pk items with href attributes
When rendered
Then each item is wrapped in a link; the last item has aria-current="page"

Given a breadcrumb-pk
When component initializes
Then data-ready="true" is set on the host
```

## Technical Notes
- Slot: `default` — `<a href="...">Label</a>` or `<span>Label</span>` items
- Script reads slotted items, wraps in `<li>` elements, marks last item with `aria-current="page"`
- CSS separator via `::after` pseudo-element using content property (CSS-only, no extra DOM)
- `nav` landmark with `aria-label="Breadcrumb"` (or custom `label` prop)
